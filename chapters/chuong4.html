Thực hành: Dựng biệt thự & nhà phố

Dựng hình công trình hoàn chỉnh từ bản vẽ AutoCAD — tường, sàn, cửa, mái, vật liệu, ánh sáng, Scenes và xuất ảnh.

Nội dung chương
  • Phần 1: Dựng biệt thự
  • Phần 2: Dựng nhà phố

Phần 1: Dựng biệt thự

A. Chuẩn bị & Import bản vẽ CAD

  1. Chỉnh đơn vị trong SketchUp: Window → Model Info → Units
  2. Import file CAD: File → Import, chọn đường dẫn tới File Cad
  3. Trước khi Import, chỉnh đơn vị trong phần Options cho khớp
  4. Chuyển về Camera → Standard Views → Top
  5. Vào Camera → Parallel Projection
  6. Zoom Extents để nhìn tổng quan bản vẽ
  7. Chọn mặt bằng tầng trệt → Click phải → Make Group
  8. Làm tương tự với mặt bằng lầu 1
  9. Lock mặt bằng trệt để không bị thay đổi khi vẽ

B. Dựng tường tầng trệt

  1. Dùng Rectangle (R) tạo mặt theo tường bao ngoài
  2. Click phải → Reverse Face để đảo ngược mặt
  3. Dùng Offset (F) với kích thước 200 (độ dày tường), xóa mặt phẳng phía trong
  4. Làm tương tự với tường bên và tường bên trong
  5. Tạo Layer: vào Window → Layers, thêm "TUONG TRET"
  6. Click đôi vào mặt tường → bảng layers → chọn TUONG TRET
  7. Click phải → Make Group
  8. Chuyển về Isometric view → Push/Pull (P) tường cao 3000mm

C. Tạo cửa đi & cửa sổ

  1. Bật chế độ X-ray (Face Style) để thấy bản vẽ CAD bên dưới
  2. Dùng Tape Measure (T) tạo đường gióng guides: chiều cao cửa 2000mm, bệ cửa sổ 1200mm
  3. Tạo guides thẳng đứng theo mặt bằng CAD
  4. Dùng Rectangle vẽ cửa theo các đường guides
  5. Dùng Push/Pull để khoét lỗ cửa xuyên qua tường (200mm)
  6. Vào Edit → Delete Guides để xóa đường gióng
💡 Mẹo khoét cửa
Khi dùng Push/Pull khoét cửa, có thể dùng chế độ gióng điểm "On Edge" để Push/Pull chính xác đến mặt bên kia của tường, hoặc nhập giá trị 200 (độ dày tường) trong VCB.

D. Dựng sàn lầu 1 & tường lầu 1

  1. Dùng Rectangle vẽ hình chữ nhật bao quanh tường → Push/Pull xuống 100mm → Make Group
  2. Tạo layer "SAN LAU 1"
  3. Vẽ tường lầu 1 tương tự tầng trệt → Push/Pull cao 3000mm → Group → Layer "TUONG LAU 1"
  4. Tạo cửa tương tự: X-ray → Tape Measure → Rectangle → Push/Pull khoét
  5. Tạo mái: Rectangle → Push/Pull 100mm → Group → Layer "SAN MAI"
  6. Dùng Move chồng tường + sàn lầu 1 lên tường trệt

E. Chi tiết cửa đi

  1. Dùng Rectangle + Line vẽ khung cửa → Push/Pull 2000mm
  2. Dùng Offset mặt phẳng vào 50mm → Push/Pull khoét lỗ
  3. Make Group khung cửa
  4. Vẽ pano kính: Rectangle → Push/Pull 10mm → Make Group → Move vào giữa khung
  5. Tạo layers: "KHUNG CUA" và "KINH"
  6. Make Component cả bộ cửa → Copy ra cho các vị trí khác
  7. Để chỉnh sửa cửa copy riêng: Click phải → Make Unique

F. Ốp vật liệu

  1. Mở Window → Materials
  2. Kính: Chọn mục Translucent → chọn ô màu thứ 2 → click vào pano kính
  3. Tường: Chọn Colors → chọn Color_000 (trắng) → click tường
  4. Khung cửa: Chọn Metal → chọn ô đầu tiên
  5. Sàn ngoài: Chọn Wood → ốp → vào Edit chỉnh kích thước
  6. Click phải → Texture → Position → Rotate 90° để xoay Map vật liệu
  7. Đá ốp: Chọn Stone → browse ảnh Map khác nếu cần
  8. Bể bơi: Wood cho thành, Tile cho vành, Water cho mặt nước
  9. Nền cỏ: Chọn Vegetation → chỉnh tỉ lệ trong Edit

G. Ánh sáng & Scenes

  1. Mở Window → Shadow: Display Shadow, chọn Time & Date
  2. Chỉnh Light (cường độ sáng), Dark (cường độ tối), On face, On ground
  3. Thiết lập Location trong Window → Model Info → Location
  4. Chuyển về Camera → Perspective
  5. Mở Window → Scenes → click "+" thêm Scene mới
  6. Tạo nhiều Scene ở các góc nhìn khác nhau

H. Xuất ảnh

  1. Vào File → Export → 2D Graphic...
  2. Chọn định dạng file trong Export type
  3. Click Options: bỏ chọn "Use view size" → nhập W × H lớn
  4. Bật Anti-alias (khử răng cưa)
  5. JPEG Compression: chọn Better (max)
  6. Đặt tên, chọn vị trí → Click Export

Phần 2: Dựng nhà phố

A. Chuẩn bị bản vẽ CAD

⚠️ Xử lý bản vẽ CAD trước khi Import
Cần 1 bản vẽ thật sạch, gọn nhẹ — không có Dim, Text, Hatch:
1. Dùng lệnh Filter trong CAD để lọc và xóa Dimension, Hatch, Text, Leader…
2. Xoay MB sao cho lối vào chính hướng xuống phía dưới.
3. Gióng mặt đứng cho chính xác với mặt bằng.
4. Cho tất cả về Layer 0, Purge bản vẽ cho sạch.
5. Import vào SketchUp: chọn Scale trong Options = mm.

B. Quy trình dựng nhà phố

  1. Import file CAD → Group mỗi mặt bằng & mặt đứng
  2. Đưa bản vẽ về gần gốc tọa độ → Xóa layer CAD thừa, chỉ giữ Layer 0
  3. Xoay mặt đứng lên theo phương thẳng đứng
  4. Tạo các Layer: CAD, Tường 1, Sàn 1, Cửa, Mái...
  5. Vẽ tường bao ngoại thất bằng Line → Offset 200mm → Reverse Faces
  6. Push/Pull tường cao 3900mm (hoặc gióng theo mặt đứng)
  7. Tạo đường gióng cửa bằng Tape Measure → khoét cửa bằng Push/Pull
  8. Xóa Guides → Group tường → đưa vào Layer
  9. Vẽ đế tầng trệt, bậc tam cấp (cao 150mm)
  10. Vẽ tường lầu 1 tương tự → Push 3600mm → Move chồng lên trệt
  11. Gióng kích thước cửa từ mặt đứng → Rectangle → Push khoét cửa lầu 1
  12. Tạo sàn lầu 1: Push 100mm → Group → Layer
  13. Vẽ lanh tô cửa, mái ban công bằng Rectangle
  14. Vẽ joint tường: Line tạo rãnh → Push 10mm
  15. Vẽ lan can kính (10mm), thanh chống đứng, tay vịn
  16. Dùng Follow Me tạo tay vịn lan can
  17. Dựng sân thượng tương tự → Bồn hoa, lan can sắt
  18. Tạo mái → Group → Layer → Move lên vị trí

C. Cửa đi nhà phố

  1. Dùng Rectangle + Line tạo khung cửa → Push/Pull 200mm → Make Group → Layer CUA DI
  2. Hide layer CUA DI → vẽ cánh cửa dễ dàng
  3. Rectangle vẽ cánh cửa → Push/Pull 50mm
  4. Cánh cửa dưới: Make Component; cánh phía trên: Make Group
  5. Vẽ Pano cửa: Rectangle → Push/Pull 5mm → Offset vào 20mm → Line chéo → Move 5mm → Push/Pull 5mm
  6. Make Component Pano → dùng Move + Ctrl copy → nhập x4 VCB
  7. Hiện layer CUA DI → Move cánh cửa + pano vào đúng vị trí

D. Ốp vật liệu nhà phố

  • Tường: Colors → màu trắng (Color_000) → ốp cho tất cả tường trước
  • Tường lầu 1: Color_C16 (màu đậm) cho tường lồi
  • Khung cửa & cánh cửa: Wood → ô thứ 4 hàng trên
  • Pano kính: Translucent → ô thứ 2
  • Nền sàn: Stone → ô thứ 1 → chỉnh kích thước trong Edit
  • Lan can sắt: Metal → ô thứ 3 hàng dưới

E. Thiết lập & Xuất ảnh

Thiết lập ánh sáng, Styles, Scenes và xuất ảnh tương tự như phần biệt thự. Vào File → Export → 2D Graphic.
