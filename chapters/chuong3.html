Chương 3

Bài tập thực hành các công cụ đã học

Thực hành dựng hình bộ phận kiến trúc — ví dụ dựng hình 1 cánh cửa đi từ bản vẽ AutoCAD.

Mục tiêu

Trình bày các ví dụ dựng hình bộ phận kiến trúc, kết hợp các công cụ đã học ở Chương 2: Line, Rectangle, Push/Pull, Offset, Move, Rotate, Scale, Follow Me, Group, Component.

Ví dụ: Dựng hình 1 cánh cửa đi

Bước 1: Import bản vẽ AutoCAD

  1. Vào File → Import, chọn file AutoCAD
  2. Click Options → thiết lập đơn vị, tỉ lệ đúng → Click OK
  3. Click Open → xuất hiện bảng Import Results → click Close
📌 Lưu ý Import CAD
Trước khi Import, cần chỉnh đơn vị trong phần Options cho khớp với bản vẽ CAD. Sau khi Import, Group từng bộ phận cánh cửa lại trước khi thao tác vẽ.

Bước 2: Chuẩn bị

  1. Group từng bộ phận cánh cửa lại
  2. Dùng Rotate xoay cánh cửa dọc theo trục Z
  3. Vào Style chỉnh nét hiển thị mảnh cho dễ vẽ

Bước 3: Dựng khung ngoại cửa

  1. Sử dụng công cụ Line, Rectangle để vẽ khung ngoại
  2. Dùng Push/Pull → nhập độ dày 250mm
  3. Click chọn khung vừa vẽ → Make Group
  4. Click chọn Hide khung cửa cho dễ vẽ các bước tiếp theo

Bước 4: Vẽ cánh cửa

  1. Xóa các mặt không cần thiết
  2. Push/Pull khung cửa dày 40mm
  3. Vẽ khung cửa phía trên tương tự
  4. Group cánh cửa lại → chọn Hide

Bước 5: Vẽ các Pano cửa

  1. Vẽ đường bao của khung cửa → Push/Pull chiều dày 5mm
  2. Vẽ đường bao trong → nối các đường chéo góc
  3. Dùng Move đưa mặt phẳng chọn ra 5mm
  4. Push/Pull mặt phẳng vừa Move ra thêm 5mm
  5. Làm tương tự cho mặt sau
  6. Group Pano cửa → Copy ra cho các vị trí khác.

Bước 6: Lắp ráp

  1. Vào Edit → Unhide → All để hiện lại tất cả các bộ phận
  2. Move các Pano cửa vào giữa cánh cửa
  3. Vẽ thêm nẹp che khung cửa với tường
  4. Vẽ đường dẫn xung quanh khung cửa
  5. Sử dụng Follow Me để trượt hình nẹp theo đường dẫn
  6. Make Component nẹp → Copy → dùng Scale để đối xứng
💡 Kết hợp Component + Scale
Khi cần tạo nẹp đối xứng: Make Component nẹp ban đầu → Copy ra → dùng Scale kéo grip giữa ngược chiều đến khi VCB hiển thị "-1" để tạo phép đối xứng hoàn hảo.

Bài tập bổ sung 1: Dựng ngôi nhà đơn giản

Bài tập cơ bản giúp làm quen với các công cụ chính: Rectangle, Push/Pull, Line, Move, Arc, Eraser, Paint Bucket.

Bước 1: Tạo khối hộp

  1. Vẽ Rectangle trên mặt đất
  2. Dùng Push/Pull kéo lên thành khối hộp (tường nhà)

Bước 2: Vẽ mái nhà

  1. Dùng Line vẽ đường nối 2 Midpoint (trung điểm) trên mái
  2. Dùng Move kéo đường vừa vẽ lên trên (theo trục xanh dương) để tạo mái dốc

Bước 3: Cửa đi

  1. Vẽ Rectangle trên mặt trước → tạo hình cửa
  2. Dùng Arc vẽ cung tròn ở đỉnh cửa → tạo cửa vòm
  3. Dùng Eraser xóa cạnh ngang trên → hợp nhất thành 1 mặt cửa
  4. Click phải vào mặt cửa → Erase để tạo lỗ nhìn xuyên qua nhà

Bước 4: Cửa sổ

  1. Vẽ Rectangle trên mặt bên → tạo cửa sổ
  2. Có thể dùng Line vẽ hình dạng bất kỳ — đảm bảo tooltip "On Face" khi vẽ
  3. Kết thúc hình tại điểm bắt đầu — đường viền sẽ trở nên mảnh khi đúng

Bước 5: Tô màu & vật liệu

  1. Click Paint Bucket (B) → mở cửa sổ Materials
  2. Chọn thư mục "Roofing" → chọn vật liệu mái → click từng mặt mái
  3. Chọn thư mục "Bricks and Cladding" → tô tường
  4. Chọn thư mục "Translucent" → tô kính cửa sổ (trong suốt)
  5. Vẽ Line chia mặt kính → tô nhiều màu Translucent tạo cửa sổ nhiều màu
💡 Mẹo thêm chi tiết
Cửa đi: Vẽ lại cạnh cửa bằng Line → mặt cửa xuất hiện trở lại → tô vật liệu gỗ từ "Wood" → thêm Circle làm mắt cửa.
3D Text: Dùng Tools → 3D Text để in tên/số nhà lên mặt tiền.

Bài tập bổ sung 2: Tòa nhà với Component

Bài tập trung bình — thực hành Component, Copy Array, Make Unique. Dựng tòa nhà cao tầng với hệ cửa sổ tròn lặp lại.

Bước 1: Dựng khối nhà

  1. Dựng khối hộp cao (tòa nhà)
  2. Dùng Arc vẽ nửa vòng tròn trên mái (đảm bảo vẽ theo hướng trục xanh dương)
  3. Push/Pull hình cung sang phía đối diện → tạo mái vòm
  4. Dùng Move kéo 1 cạnh giữa tường → chia tường thành 2 phần: thẳng đứng và nghiêng

Bước 2: Tạo Component cửa sổ

  1. Tô Paint Bucket cho mặt tường
  2. Vẽ 2 Circle đồng tâm tại góc tường → tạo cửa sổ tròn có khung
  3. Select cả 2 mặt → Click phải → Make Component
  4. Đặt tên, check "Cut Opening" và "Replace selection with component"
📌 Cut Opening
Thuộc tính Cut Opening khiến component tự khoét lỗ khi đặt lên mặt phẳng — hoàn hảo cho cửa sổ và cửa đi. Khi copy component sang mặt khác, lỗ cũng tự tạo.

Bước 3: Copy dãy cửa sổ

  1. Dùng Move + Ctrl → copy cửa sổ sang bên phải
  2. Nhập x3 → Enter → tạo dãy 4 cửa sổ ngang
  3. Select cả dãy → Move + Ctrl copy lên trên → nhập x2 → tạo thêm 2 hàng
  4. Copy một số cửa sổ lên mặt nghiêng — chúng tự cắt mặt nghiêng

Bước 4: Chỉnh sửa đồng loạt + Make Unique

  1. Double-click vào bất kỳ cửa sổ nào → vào chế độ chỉnh sửa component
  2. Dùng Push/Pull đẩy vòng ngoài → tạo khung nổi
  3. Tô màu → Tất cả cửa sổ tự cập nhật
  4. Click ngoài để đóng → Chọn các cửa sổ trên mặt nghiêng → Click phải → Make Unique
  5. Chỉnh sửa cửa sổ Unique: thêm mái che bằng Push/Pull + Ctrl
💡 Push/Pull + Ctrl
Giữ Ctrl khi Pull sẽ tạo phần mới thay vì kéo mặt phẳng hiện tại. Rất hữu ích khi thêm chi tiết (mái che, gờ, khung cửa) mà không ảnh hưởng hình dạng gốc.

Bài tập bổ sung 3: Khung tranh Gallery

Thực hành Import Image, Explode, Push/Pull, Offset, Follow Me — dựng tranh có khung treo trong phòng trưng bày.

📌 Nguồn tham khảo
Bài tập dựa trên "Art / History Project: Grades 6–9" trong SketchUp Teacher Guide (3DVinci).
  1. Tìm ảnh tranh trên Internet (Google Image Search) → lưu file JPG/PNG
  2. Mở SketchUp file mới → File → Import → Use as image → đặt ảnh lên mặt đất
  3. Click phải → Explode để biến ảnh thành Face
  4. Dùng Push/Pull kéo mặt ảnh lên ~5mm
  5. Chọn mặt sau → Offset ra ngoài ~15mm → tạo viền khung
  6. Dùng Push/Pull kéo viền khung ra ~10mm → tô vật liệu gỗ từ "Wood"
  7. (Nâng cao) Vẽ tiết diện phào chỉ → dùng Follow Me trượt theo viền khung
  8. Vào Window → Model Info → File → đặt Glue to: Any
  9. Lưu file (mỗi bức tranh 1 file riêng) → File → Save As
  10. Mở file phòng trưng bày → File → Import (chọn .skp) → đặt tranh lên tường
💡 Mẹo trình bày Gallery
Dùng Walk, Look Around, Position Camera để "đi bộ" xem tranh trong gallery → lưu từng góc nhìn thành Scene (Window → Scenes) → chạy animation bằng View → Animation → Play.

Bài tập bổ sung 4: Cầu thang từ Component

Thực hành Component, Move Copy / chia đều, Follow Me — dựng cầu thang có lan can từ 1 bậc thang component.

  1. Vẽ Rectangle nhỏ (~120cm × 2cm) dọc theo tường → đây là bậc thang đầu tiên
  2. Select mặt + cạnh → Click phải → Make Component (đặt tên "Step")
  3. Double-click vào component → dùng Push/Pull kéo bậc thang dài ~100cm
  4. Đóng component → dùng Move + Ctrl copy bậc thang đầu lên đầu cầu thang
  5. Nhập /9 → Enter → SketchUp tự chia đều 10 bậc giữa 2 điểm
  6. Double-click vào bất kỳ bậc nào → vẽ đường Line thẳng đứng (3 feet) tại góc bậc
  7. Vẽ thêm Line nối đỉnh cột thẳng đứng sang bậc kế tiếp (đường chéo lan can)
  8. Vẽ Circle nhỏ tại chân cột → Select 2 đường Line
  9. Kích hoạt Follow Me → click vào Circle → ống lan can xuất hiện tại tất cả bậc thang
  10. Tô vật liệu Metal cho lan can, Wood cho bậc thang
💡 Sức mạnh Component
Vì mọi bậc thang đều là component giống nhau, chỉ cần vẽ lan can trong 1 bậc → tất cả bậc khác tự tạo lan can tương tự. Đây là lợi ích lớn nhất của Component so với Group!
